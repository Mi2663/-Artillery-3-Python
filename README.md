# Artillery 3 - Портирование с BASIC на Python

## Описание проекта
Портирование классической игры "Artillery 3" с языка BASIC на современный Python с поддержкой множества платформ.

## Особенности реализации
- Полное сохранение оригинальной игровой механики
- Автоматическое тестирование с сравнением с оригиналом
- Кросс-платформенная совместимость
- Чистый, документированный код

## Требования
- Python 3.8 или выше
- pip для установки зависимостей
