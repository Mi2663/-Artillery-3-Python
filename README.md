# Artillery 3 (WAR3) - Python Port
Точный перевод классической артиллерийской игры из "More BASIC Computer Games" (1979) на современный Python.

## Особенности

- ** Точная копия** оригинальной игровой логики и диалогов.
- ** Кроссплатформенная ** поддержка (Windows, Linux, macOS, Web, Android через Termux)
- **Автоматическое тестирование ** для обеспечения совместимости с оригинальным поведением
- ** Простая система сборки ** с Makefile

## Как играть

Два или три игрока по очереди стреляют друг в друга из пушек. Сначала вы устанавливаете расстояние между игроками и начальную скорость выстрела, затем по очереди вводите углы обстрела (в градусах), чтобы попытаться поразить своих противников.

Попадание засчитывается, если ваш выстрел попадает в цель на расстоянии менее 5% от расстояния до цели.

## Требования

- Python 3.6 или выше
- Никаких внешних зависимостей (используется только стандартная библиотека)

## Установка и запуск

### Быстрый запуск
```bash
# Клонировать репозиторий
git clone https://github.com/yourusername/artillery3-python.git
cd artillery3-python

# Запустите игру
make run
# или напрямую:
python3 artillery3.py
